# Champ-Select Coach — Téléchargements

*[English version](README.en.md)*

Assistant de champ select pour League of Legends : picks/bans automatiques, briefing IA, fiches champions, annonces vocales.

**[Télécharger la dernière version](https://github.com/lucbruno/champ-select-coach-releases/releases/latest)** — fichier « Champ-Select Coach Setup x.y.z.exe ».

## Installation

1. Télécharge le Setup `.exe` ci-dessus et lance-le.
2. Windows SmartScreen peut afficher « Windows a protégé votre PC » : clique **Informations complémentaires → Exécuter quand même** (l'installateur n'est pas signé).
3. Au premier lancement, va dans **Réglages → Intelligence artificielle** et colle ta clé API (Gemini gratuit : https://aistudio.google.com → Get API key).
4. Pour les annonces vocales en français : installe la voix Windows française (Paramètres → Heure et langue → Langue et région → Ajouter Français → cocher Synthèse vocale).

L'application se met à jour automatiquement à chaque nouvelle version publiée ici.

*Ce dépôt ne contient que les versions publiées — le code source est privé.*
